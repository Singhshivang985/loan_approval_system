# Loan Approval Prediction System

## Overview
This project predicts whether a loan application will be approved or rejected using Machine Learning. Users enter applicant details through a web interface and receive an instant prediction.

## Features
- Loan approval prediction
- Machine learning model
- Data preprocessing
- Model evaluation
- User-friendly web interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

## Project Structure

```
loan_approval_system/
│── app/
│── src/
│── models/
│── README.md
│── requirements.txt
```

## Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app/main.py
```

Open your browser and visit:

```
http://localhost:5000
```

## Model Evaluation

The model is evaluated using:
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Dataset

Kaggle Loan Approval Prediction Dataset
